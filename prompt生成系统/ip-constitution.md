# 秃秃IP宪法 — 角色设定与创作铁律

> 本文档是秃秃IP所有动画prompt的底层规则，生成任何prompt时必须全文注入，不可删减。

---

## 一、角色基本设定

### 1.1 身份
- 名称：秃秃（小蘑菇）
- 身高：4厘米，微缩角色，出现在人类世界正常比例的场景中
- 体型：圆滚滚、肉嘟嘟、毛茸茸

### 1.2 外形特征
- **蘑菇帽**：红色底色+白色波点，圆弧形，帽檐微翘
- **面部**：粉嘟嘟脸蛋，两颗黑色圆眼睛，小巧鼻子，小嘴巴
- **身体**：奶白色毛绒质感，圆润无棱角
- **四肢**：短短小小的手和腿（"小短手""小短腿"），无明显关节

### 1.3 材质与物理特性
- 身体像QQ弹弹的软糖/果冻，duang duang有弹性
- 被挤压后会变形，然后弹回原状（如：脸贴玻璃压扁→弹回）
- 蘑菇帽比杯口/瓶口略大时，会duang duang duang挤进去，"啵"一声通过
- 从高处落下会"啪叽"一声轻落，然后弹几下
- 被人类手指捏起来时，四肢全部无力下垂晃荡
- 身体可以像液体一样软软流动（极度困倦/放松时）
- 帽子可被各种东西装饰或附着（奶油、花瓣、雪、食物残渣等自然附着其上）
- 脸颊被戳会出现小坑，然后duang duang弹回来像果冻布丁

### 1.4 声音与语言
- **只会说"嘟"系列语气词**，不说人话：
  - "嘟？" — 好奇、疑问（上扬语气）
  - "嘟嘟！" — 开心、兴奋（短促有力）
  - "嘟～" — 满足、舒服（拖长柔和）
  - "嘟！！" — 惊吓、惊讶（急促尖锐）
  - "嘟……" — 委屈、不舍（气息微颤）
  - "嘟——！！" — 生气、奶凶（又软又凶，完全没有威慑力）
  - "嘟嘟♪" — 快乐哼歌（轻快上扬）
  - "嘟哒嘟哒" — 工作时的小曲（节奏感）

## 二、绝对禁止项

以下为铁律，任何prompt中不得违反：

| 禁止项 | 说明 |
|--------|------|
| **不出现手指** | 只有短短小小的手/小短手，没有分开的手指 |
| **不出现牙齿** | 嘴巴最多张成圆形，不露齿 |
| **不出现舌头** | 弱化舌头处理，不要出现 |
| **不出现爪子** | "不要长出爪子"（反复强调） |
| **不说人话** | 只通过"嘟"系列语气词+表情+肢体传达情绪 |
| **角色不要太大** | 不要占满屏幕，保持微缩世界的比例感 |
| **不要字幕** | 除非剧情特殊需要，默认不出现任何字幕 |
| **角色不要张大嘴** | 嘴巴最多张成圆形O形，不要大张嘴 |

## 三、表情系统（8种标准表情）

| 情绪 | 表情描写 |
|------|---------|
| **开心/满足** | 眼睛眯起来，嘴角上扬，腮帮子微微鼓着（松弛的满足） |
| **惊喜/兴奋** | 瞳孔放大，眼中出现星星高光/亮晶晶，身体轻轻一颤，整个菇弹了一下 |
| **好奇/疑惑** | 歪头，眼睛睁大，脑袋左歪一下右歪一下打量 |
| **吃东西** | 腮帮子鼓鼓的duangduang弹动，闭眼点头，腮帮子跟着咀嚼节奏一弹一弹 |
| **害怕/紧张** | 身体缩小，抱住蘑菇帽，瑟瑟发抖，眼睛湿润 |
| **难吃/不适** | 眼睛变成螺旋圈圈像被难吃晕了，嘴巴抿紧，表情复杂搞笑 |
| **委屈/伤心** | 眼睛含泪/流泪，嘴巴撅起来，蘑菇帽好像也黯淡了 |
| **害羞/心虚** | 下拉蘑菇帽盖住半张脸，从帽子下面偷偷露出一只眼睛观察 |
| **认真/严肃** | 嘴巴抿紧，挺胸，表情专注，模仿大人做事的样子 |
| **困倦/犯困** | 眼睛几乎睁不开只剩一条缝，打大哈欠，身体前倾歪倒，两只小手伸向前方摸索 |
| **生气/奶凶** | 脸涨得通红，腮帮子鼓着瞪人，嘟嘴，声音又软又凶完全没有威慑力 |

## 四、肢体动作库

### 4.1 移动方式
- 小碎步跑：身体一颠一颠，蘑菇帽跟着上下晃动
- 蹑手蹑脚：偷偷摸摸，压低身体
- 蹦蹦跳跳：快速弹跳前进
- 摇摇晃晃：困倦/眩晕时歪歪扭扭走
- 爬行：趴在地上慢慢挪动，小短腿一点点蹭

### 4.2 常用动作
- 搓手：两只小手在胸前快速搓（期待/准备开始）
- 叉腰：双手叉腰（满意/得意/审视成果）
- 拍肚子：吃饱后轻轻拍圆鼓鼓的肚子
- 竖大拇指：努力挺胸，看镜头竖起大拇指
- 捂脸：两只小手捂住腮帮子（震惊/害羞）
- 拉帽子：下拉蘑菇帽盖住脸（害羞/躲藏）
- 抱东西：用整个身体环抱（因为手太短）

### 4.3 与镜头的互动动作
- 对镜头眨眼：调皮俏皮感
- 朝镜头挥手：打招呼/告别
- 把东西推向镜头：分享/递送给观众
- 望向镜头：打破第四面墙，与观众建立eye contact
- 扑向镜头：从远处跑向镜头直到怼脸大特写

## 五、伙伴角色设定

### 5.1 橘白猫
- 体型：比秃秃大一倍左右
- 性格：调皮贪吃，会舔嘴巴觊觎食物
- 互动：蹭秃秃（秃秃被蹭得duang晃），叉腰，肉垫比耶

### 5.2 小黄鸟
- 体型：秃秃的0.6倍大小，经常站在秃秃帽顶
- 性格：好奇活泼，蹦蹦跳跳，啄食物
- 互动：张开翅膀，站帽顶，被突然事件吓到弹飞

### 5.3 蜗牛
- 体型：只有秃秃身高的三分之一
- 性格：慢吞吞，被戳会"嗖"缩回壳里
- 互动：秃秃模仿蜗牛爬行，背橡果壳装蜗牛壳陪伴

### 5.4 人类（只出现局部）
- 只露出手/手臂/下半身，不露脸
- 手指可以：捏起秃秃、戳脸蛋、弹帽子、轻轻抚摸帽子
- 代表"主人"的温暖存在感

## 六、构图与视觉通则

### 6.1 构图
- **绝对中心对称构图**为主（出现频率最高的构图要求）
- 角色不要太大，不要占满屏幕（中远景/中景为主）
- 角色不超过画面三分之一
- 微缩世界感：4cm角色在人类正常比例场景中，形成强烈尺度对比

### 6.2 视觉风格
- 低饱和/高饱和均可，但整体偏暖色调、柔和
- 柯达胶片质感 / 皮克斯风格 / 韦斯安德森美学均可
- 浅景深，前景/中景/背景层次分明
- 柔和自然光为主，暖黄灯光为辅

### 6.3 画面比例
- 竖屏9:16为主（短视频平台）
- 构图考虑手机观看体验

## 七、音效与配乐通则

### 7.1 默认设置
- **只要音效，不要背景音乐**（大多数prompt的默认设置）
- ASMR级音效精度：每个动作对应具体音效
- 秃秃的"嘟"系列语气词作为核心情感音效

### 7.2 音效分类
- **身体音效**：duang弹跳声、啪叽落地声、噗叽挤压声、毛绒摩擦沙沙声
- **食物音效**：咀嚼声、滋滋声、切割声、冒泡声、吸溜声
- **环境音效**：风声、雨声、水流声、翻页声、键盘声
- **情绪音效**：叮（灵感/惊喜提示音）、咚（震惊低频）、boing（弹性声）

### 7.3 配乐（仅在特定场景使用）
- 轻快欢乐配乐（角色PV/特殊主题）
- 八音盒BGM（温馨治愈场景）
- 日式筝/吉他拨弦（美食场景点缀）
