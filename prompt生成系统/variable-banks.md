## 5. 体型喜剧池

### 被卡住系列
- 蘑菇帽卡在杯口/瓶口 → duang duang挤进去 → 啵一声通过
- 钻进鸡蛋格/抽屉/盒子 → 被当成其他东西
- 藏在食物堆里 → 被误认为食材
- 被夹在书页中间 → 翻不动页

### 坠落系列
- 从桌沿/食物上滑下 → 啪叽一声 → 弹几下
- 一脚踩空掉进杯子/碗里 → 卡在杯口再掉进去
- 被弹射到空中 → 在空中翻滚 → 软着陆
- 从面包片楼梯上像液体一样滑下去

### 被弹飞系列
- 被扫地机器人撞飞 → 在空中"嘟——！！"
- 坐在弹力物上 → 意外被弹射
- 被人类动作的气流吹飞
- 被猫尾巴扫飞

### 变形系列
- 脸贴玻璃/秤面压扁 → 弹回duang
- 被捏起来四肢无力下垂
- 整个身体像液体流下去（困倦）
- 被挤压变成扁饼 → 慢慢恢复

### 攀爬系列
- 爬杯沿/碗沿 → 翻上去趴一下
- 爬食物堆（面包楼梯/水果山）
- 够不到的东西 → 踮脚 → 跳 → 还是够不到
- 叠东西当梯子 → 摇摇晃晃

### 比例反差系列
- 食物比秃秃大（一片薯片=秃秃的脸大小）
- 工具太大太重（筷子像两根长棍）
- 近大远小（近景霸气→远景迷你可笑）
- 人类的一根手指 = 秃秃的身高

## 7. 运镜池

| 运镜方式 | 描述 | 适用场景 |
|----------|------|---------|
| **固定中景平拍** | 最常用，中景固定机位，正面平视 | 美食制作/吃播 |
| **绝对中心对称** | 水平绝对对称构图 | 所有类别通用 |
| **镜头跟随** | 镜头跟随秃秃移动 | 移动/探索场景 |
| **POV切换** | 第三人称→秃秃POV→观众POV | 吃播/互动 |
| **仰拍低角度** | 贴地与秤面/地板同高 | 强调微缩感 |
| **俯拍45度** | 从上方看秃秃 | 人类/动物视角 |
| **推近特写** | 从中景推到脸部/食物特写 | 表情/食物细节 |
| **一镜到底** | 丝滑连贯不切镜 | 角色PV/流畅动作 |
| **手持晃动** | 模拟手机拍摄POV | 人类视角发现秃秃 |
| **拉远定格** | 镜头缓慢拉远→画面定格 | 温馨/诗意收尾 |
| **快速跳切** | 每个动作快速切换 | 制作过程/连续动作 |
| **从内部往外拍** | 从容器/冰箱内部拍 | 秃秃被发现/POV |
