# 6大类别模板 — 结构骨架与变量槽位

> 每个模板定义了该类别prompt的固定结构、情绪弧线公式、必填变量槽位。
> `{变量}` 表示需要填入的创意变量，从 variable-banks.md 中选取或自由创作。

---

## 模板A：美食制作类

**情绪弧线：** 准备食材 → 认真制作 → 成品展示 → 反转收尾（偷吃/搞砸/分享）

**结构骨架：**

```
【角色形象参考】图片1是小蘑菇角色的设定和各个形态参考图。
【构图】绝对中心对称构图，蘑菇角色不要太大，不要超过画面三分之一。
【场景】{场景描述}，场景参考图片2。
【运镜】运镜丝滑，{运镜风格}。
【氛围】氛围欢快温馨。

---分段时间线---

第一段（准备阶段）：
（中景｜固定）：蘑菇角色{准备动作描述}。
{食材质感描写——具体到颜色、层次、状态变化}。
只要音效，不要背景音乐，不要有字幕，不要长出爪子

第二段（核心制作）：
（中景｜固定）：蘑菇角色{核心制作动作}。
{制作过程中的质感变化——颜色渐变、形态变化、温度变化}。
只要音效，不要背景音乐，不要有字幕，不要长出爪子

第三段（成品展示）：
{成品的视觉描写——切面/外观/颜色层次/质感}。
蘑菇角色{装饰/摆盘动作}。
只要音效，不要背景音乐

第四段（品尝/收尾）：
蘑菇角色{品尝动作}，腮帮子{咀嚼反应}。
{反转/搞笑/温馨收尾}。
{互动beat——递向镜头/满足地看向镜头}
只要音效，不要背景音乐
```

**必填变量：**
- `{场景描述}`: 厨房类型（中式厨房/户外烧烤/街边小摊/面包工坊...）
- `{食物名称}`: 具体食物
- `{食材质感描写}`: 至少包含3种颜色+2种质感词
- `{核心制作动作}`: 主要制作步骤（搅拌/揉面/烤制/切割/编织...）
- `{成品视觉}`: 切面/外观的逐层颜色描写
- `{反转收尾}`: 偷吃产品/搞砸一步/把成品推向镜头

## 模板B：美食吃播类

**情绪弧线：** 发现食物 → 好奇打量 → 试探第一口 → 表情炸裂 → 大快朵颐 → 收尾

**结构骨架：**

```
【角色形象参考】图片1是小蘑菇角色的设定和各个形态参考图。
【构图】绝对中心对称构图为主，蘑菇角色不要太大。
【场景】{场景描述}，场景参考图片2。
【运镜】运镜丝滑，氛围欢快温馨。

0-{t1}s：{发现食物}。蘑菇角色{接近方式——从后面探出脑袋/鬼鬼祟祟冒出/好奇走近}。
{食物的视觉描写——逐层/逐色/逐质感，至少描写3层以上}。

{t1}-{t2}s：蘑菇角色{试探动作——闻一闻/歪头打量/伸手沾一点}。
表情从好奇变成{第一反应——期待/犹豫/跃跃欲试}。

{t2}-{t3}s：第一口{吃法——咬/舔/挖/吸溜/塞进嘴里}。
{第一口反应——眼睛一亮/星星高光/腮帮子duangduang鼓起来}。
第二口{升级吃法}，嚼得更快更开心。
第三口{更夸张的吃法}，腮帮子鼓到最大。

{t3}-{t4}s：{收尾——吃完了摸肚子/发现没了很失落/被人发现偷吃}。
{互动beat——望向镜头满足一笑/把最后一口递向镜头}

只要音效，asmr，不要背景音乐，不要字幕。
```

**必填变量：**
- `{食物}`: 具体食物（含视觉描写）
- `{接近方式}`: 如何发现/靠近食物
- `{食物逐层描写}`: 从上到下/从外到内，每层颜色+质感
- `{吃法}`: 筷子夹/勺子挖/直接啃/吸溜/手抓
- `{每口反应递进}`: 第一口→第二口→第三口的表情变化升级
- `{收尾反转}`: 吃撑/被发现/食物没了/分享

## 模板C：日常生活类

**情绪弧线：** 发现人类物品 → 好奇互动 → 体型差异导致意外 → 情绪解决

**结构骨架：**

```
【角色形象参考】图片1是小蘑菇角色的设定和三视图。
【构图】绝对中心对称构图为主。
【场景】{家居场景描述}，场景参考图片2。
【运镜】运镜丝滑，氛围欢快温馨。
【光线】{时间段}光线，{光线质感}。

0-{t1}s：{环境建立}。
蘑菇角色{发现物品的方式——看到人在用/自己偶然发现/被声音吸引}。
歪头{观察反应}，眼睛跟着{物品的运动}移动。
动作：{灵感一闪/决定尝试/忍不住了}。

{t1}-{t2}s：蘑菇角色{用自己的方式与物品互动}。
{体型差异带来的物理喜剧——物品对它来说太大/太重/太高/比例不对}。
{互动过程中的身体反应——duang弹/被弹飞/卡住/滑倒}。

{t2}-{t3}s：{高潮/意外发生}。
{因体型产生的搞笑结果}。
蘑菇角色{意外后的反应——愣住/眩晕/假装没事/得意}。

{t3}-{t4}s：{温馨收尾}。
{互动beat——看向镜头竖大拇指/靠着物品满足地待着/对镜头"嘟～"}
只要音效，不要背景音乐，不要字幕。
```

**必填变量：**
- `{人类物品}`: 体重秤/键盘/扫地机器人/熨斗/沙发...
- `{家居场景}`: 客厅/厨房/卧室/阳台/浴室
- `{互动方式}`: 秃秃如何以蘑菇逻辑使用这个物品
- `{体型喜剧}`: 因4cm身高导致的搞笑情况
- `{情绪解决}`: 温馨/满足/得意/释然的收尾

## 模板D：第一次认识X类

**情绪弧线：** 发现未知物品 → 好奇接近 → 用蘑菇逻辑误解/误用 → 搞笑或惊吓结果

**结构骨架：**

```
【核心概念】秃秃第一次遇到{人类物品}，不知道是什么，用自己的方式去理解。
【角色形象参考】图片1是小蘑菇角色的设定和三视图。
【场景】人类{场景}环境中，偶然发现{物品}。
【运镜】流畅丝滑的运镜，有动画感。

镜头1（0-{t1}s）：{发现物品}。
蘑菇角色{好奇接近——偷偷摸摸/小心翼翼/蹦蹦跳跳跑过去}。
{物品在秃秃视角下的描写——对它来说像什么/有多大}。

镜头2（{t1}-{t2}s）：{误解/误用物品}。
蘑菇角色{用错误但合理的方式使用物品——把耳机当床/把墨镜当盾牌/把书当房子}。
{误用过程中的细节描写}。

镜头3（{t2}-{t3}s）：{搞笑/惊吓结果}。
{反转——发现书里写着吃蘑菇/耳机突然播放音乐被吓到/墨镜太重戴不住}。
蘑菇角色{最终反应}。
{互动beat——看着镜头流泪/兴奋对镜头眨眼/慌张望向镜头求助}
```

**必填变量：**
- `{物品}`: 人类日常物品
- `{误解方式}`: 秃秃对物品的蘑菇式理解
- `{反转结果}`: 搞笑/惊吓/意外发现
- `{互动收尾}`: 与观众分享这个反应

## 模板E：户外/主题场景类

**情绪弧线：** 氛围沉浸 → 缓慢探索 → 自然互动 → 诗意定格

**结构骨架：**

```
【角色形象参考】📷图片1是小蘑菇的设定和三视图，小蘑菇只有4cm高，构图均为中近景，角色人物不要太小。
【场景】📷图片2这是可以参考的场景置景、风格和打光，但元素不要完全一样。
{场景的诗意描写——自然环境、光线、色彩、氛围，至少3句}。

小蘑菇角色{在场景中的位置和状态}，身体毛茸茸圆滚滚保持原本肉嘟嘟体型{服装/装饰描写，如有}。

{慢节奏的互动描写}：
- {与自然元素的互动——触碰水面/接住花瓣/感受微风/仰望星空}
- {身体细节——毛绒边缘的光晕/帽子上落了花瓣/微风吹动的披帛}
- {面部表情——安静温柔/好奇/满足微笑/闭眼享受}

{诗意收尾——画面定格在某个美好瞬间}。
{互动beat——抬头看向镜头眼睛亮晶晶/对镜头露出温柔微笑}
没有配乐伴奏，只有音效。
```

**必填变量：**
- `{自然场景}`: 樱花树下/江南水乡/森林/花田/雪地/海边...
- `{季节/时间}`: 春日/夏夜/秋天/冬雪/黄昏/清晨
- `{自然互动}`: 与花/水/风/光/雪等自然元素的具体互动
- `{诗意定格}`: 最终画面的定格描写

## 模板G：秃秃家中类（自家童话世界）

**核心区别于其他模板：**
- 场景是**秃秃自己的家**（蘑菇尺度的童话室内空间），不是人类世界。
- 家具/物品的大小**和秃秃匹配**：圆床、唱片机、矮柜、拱形小木门、玄关矮柜、瓶盖座椅 等都是为 4cm 蘑菇身材量身打造的尺度，没有"对它来说像山一样大"的体型反差。
- 装饰元素是自然/童话风：浆果、树叶、花瓣、草绳、萤火苔藓、蘑菇灯、星座、夕阳……不出现现代人类电器。
- 节奏比 collection 类**慢、温馨、生活化**，主打"独处也很满足"的氛围。
- 必须从下面 4 张家中场景参考图里选一张作为陈设依据，并通过显式占位符 `{scene:KEY}` 引用：
  - `{scene:bedroom}`        蘑菇卧室（圆床+唱片机+蒲公英摆件+绿色矮柜，窗外蓝天）
  - `{scene:living_bedroom}` 客厅+卧室（粉色沙发+橙色台灯+Monday 日历+郁金香+床角）
  - `{scene:game_dressing}`  游戏室+衣帽间（开放衣架+四件小衣服+镜子+红色瓶盖座椅+草坪地毯+唱片机）
  - `{scene:entrance}`       玄关（拱形木门+玄关矮柜+米色羊皮地毯+粉色沙发角）

**情绪弧线：** 温馨开场 → 缓慢专注做某件小事 → 出现一点小惊喜/小波折 → 满足定格

**结构骨架：**

```
【角色形象参考】图片1是小蘑菇角色的设定和各个形态参考图。
【场景参考】参考{scene:KEY}场景图——{用一两句话复述参考图的关键陈设——XX家具+XX装饰+XX光线}。
【构图】水平中心对称构图 + 镜头固定平拍 + 低平视角贴近{家具表面/地面}。秃秃出现在该空间的{具体位置——某件家具上/旁/前/里}。
【尺度说明】秃秃和家具是匹配的童话尺度，不强调"对它来说太大"的反差。家具/装饰的尺寸只用作画面构图参考，不写成"高耸/巨大/数倍"。
【光线】{时间段}的{光线质感}（清晨柔光/正午暖光/傍晚橘色夕阳/夜晚暖黄台灯/萤火青光）。
【氛围】温馨、安静、独处的满足感，节奏慢。

0-{t1}s：{环境建立 + 秃秃出场}。
镜头从{家中某处}开始，秃秃{进入画面的方式——从XX探出来/慢慢走过来/已经在那儿}。
{他正在准备/酝酿要做的事——歪头打量手里的XX/伸懒腰}。
音效：{环境氛围声——窗外鸟鸣、风铃叮当、火苗噼啪、唱片机嗡嗡}。

{t1}-{t2}s：秃秃{开始动手做事——核心动作1}。
{动作过程中的细节——身体晃动/帽子歪斜/腮帮子鼓鼓/眼睛专注}。
{家中场景里的小元素被点亮/扰动}（如：风让风铃响、火苗跳动、苔藓泛蓝光）。
音效：{动作音效逐拍}。

{t2}-{t3}s：{出现小惊喜或小波折}。
{波折描写——东西没成功/掉了/换了形态/意外好看了}。
秃秃{应对——继续/换方式/愣一下/小骄傲}。
音效：{对应的拟声词——啪叽/duang/沙沙/叮}。

{t3}-{t4}s：{温馨定格}。
{最终画面——秃秃和成品/家具一起的安静画面，光线/氛围烘托满足感}。
{互动beat——望向镜头一声满足的"嘟～"/帽子歪着定格/慢慢眨眼}
只要音效，不要背景音乐，不要字幕。角色特征：小蘑菇的嘴巴张开时里面是黑色的小圆洞（无牙齿无舌头），身体两侧伸出的肢体末端是圆滚滚的肉球状/棉花糖状（整体一个圆球，无分开的指头，无爪状结构），没有眉毛，没有尾巴。
```

**必填变量：**
- `{scene:KEY}`: 4 选一的家中场景占位符（bedroom / living_bedroom / game_dressing / entrance）
- `{家具/位置}`: 秃秃在场景里的具体位置（绿色矮柜上 / 沙发旁 / 玄关地毯上 / 拱门口 / 阳台窗台）
- `{核心动作}`: 这条事件的主要小事（烤浆果 / 染帽子 / 拼苔藓 / 挂风铃 / 排石子 / 跳绳 / 剪纸 / 拉小提琴）
- `{自然装饰}`: 用到的童话元素（浆果 / 树叶 / 花瓣 / 草绳 / 萤火苔藓 / 露珠 / 蘑菇灯 / 唱片）
- `{光线时段}`: 清晨 / 正午 / 傍晚 / 夜晚（影响色调和氛围）
- `{小惊喜/小波折}`: 风把风铃吹响 / 帽子被染色 / 石子排歪了 / 露水滴到帽子 等
- `{诗意定格}`: 最后一帧的画面构图

**严禁：**
- 不写"对它来说像山一样大""比身体高 N 倍""巨大"等微缩反差描写
- 不出现人类电器（手机/电脑/微波炉/咖啡机等）；只用童话物件
- 不出现"NPC人类"在画面里——这是秃秃独处的家
- 见 IP 宪法的禁说话铁律（不能出现"仿佛在说X"等心理语言旁白）

## 互动Beat速查表

所有模板的收尾处都需要加入互动beat，从以下选取：

| 互动类型 | 具体动作 | 适用场景 |
|----------|---------|---------|
| 递送型 | 两只小手把东西推向镜头方向，脸涨红用力 | 美食制作/饮品 |
| 对话型 | 指了指东西又指了指镜头，歪头眨眼 | 分享成果 |
| 展示型 | 转一圈向镜头展示，叉腰满意 | 制作成品/职业成果 |
| 求助型 | 望向镜头，表情委屈/困惑 | 遇到困难/搞砸 |
