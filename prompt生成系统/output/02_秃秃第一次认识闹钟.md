# 秃秃第一次认识闹钟

类别: 第一次认识X
情绪线: 安静沉睡→被惊醒暴怒→奶凶反击→心虚求和
互动类型: 轻互动收尾

---

📷图片1是小蘑菇角色的设定和三视图，小蘑菇只有4cm高，构图均为中近景，角色人物不要太大。📷图片2这是首帧参考图，参考卧室床头柜的置景和清晨光线氛围。绝对中心对称构图，蘑菇角色不要超过画面三分之一。场景：清晨卧室床头柜上，一个圆形复古金属小闹钟放在柜子右侧，闹钟和蘑菇角色差不多高。左侧是一盏关着的小台灯。窗帘缝隙透进来淡淡的晨光，整体画面安静柔和。微距浅景深，背景虚化可见床铺和枕头轮廓。角色没有手指和牙齿。

0-3s：清晨安静的卧室，床头柜上蘑菇角色蜷在闹钟旁边熟睡，身体蜷成一小团靠着闹钟金属外壳，蘑菇帽微微歪向一边，小脸贴着冰凉的金属面，腮帮子被压得鼓出来一点点。身体一起一伏均匀呼吸。闹钟指针滴答滴答走动。安静了两秒——突然闹钟铃响了！！两个小锤子疯狂敲打顶部铃铛，整个闹钟剧烈震动。蘑菇角色被震得弹离闹钟，啪叽一声摔在柜面上弹了两下，蘑菇帽被震歪。音效：前2秒只有极轻的呼吸声"呼——嘟……呼——嘟……"和钟摆滴答声。然后铃声炸响——尖锐急促的金属铃铃铃声。蘑菇角色弹飞时"嘟！！"惊吓尖叫，落地啪叽一声。

3-7s：蘑菇角色从柜面上爬起来，蘑菇帽还歪着，眼睛瞪得圆圆的盯着闹钟——闹钟还在疯狂震动铃铃铃响，每震一下闹钟就在柜面上跳动移位一点，像一个发了疯的铁壳怪物。蘑菇角色脸涨红了，腮帮子鼓起来，嘴巴紧抿——它生气了。它气呼呼迈着小碎步冲向闹钟，两只小短手抱住闹钟顶部想按停，但闹钟震动太剧烈，蘑菇角色整个身体被闹钟带着duangduangduang上下颠，像骑着一头疯牛，两只小短腿离地悬空乱蹬。它死死抱着不松手，脸上的表情从生气变成崩溃。音效：闹钟持续铃铃铃震动声，蘑菇角色抱上去时毛绒身体蹭金属的沙沙声，被颠的时候身体duang duang duang弹动声。蘑菇角色奶凶地"嘟——！！嘟——！！"吼了两声，又软又凶完全没有威慑力。

7-10s：蘑菇角色终于找到了闹钟背面的按钮——它松开一只手，用整个身体的重量往按钮上一压——"咔哒"，铃声停了。世界突然安静。蘑菇角色趴在闹钟顶上大口喘气，蘑菇帽完全歪到一侧，身体像泄了气一样软软摊在闹钟上方。它喘了两秒，慢慢滑下来，站在闹钟旁边。低头看了看闹钟的表盘——闹钟安静地滴答走着，像什么都没发生过。蘑菇角色歪头看了看闹钟，又看了看自己刚才抱过的地方（金属面上留了一个粉色的腮红印子）。它愣了一下，伸出一只小短手轻轻拍了拍闹钟顶部，像在安抚一个脾气暴躁的朋友。音效：咔哒按钮声后突然安静，只剩蘑菇角色喘气声"嘟……哈……嘟……"和钟摆恢复的滴答声。拍闹钟顶部极轻的叮叮两声。

10-13s：蘑菇角色重新靠着闹钟坐下来，这次没有贴太近，保持了一个"安全距离"。它抱着自己的小短腿，侧头警惕地看了闹钟一眼，又看向镜头——表情又困又委屈又心有余悸，腮帮子微微撅着，蘑菇帽还歪着没扶正。它对着镜头缓慢地摇了两下头，好像在说"你不懂它有多可怕"。然后它打了个小哈欠，眼皮开始打架，头往下栽了一下——又猛地抬起来警惕地瞪了闹钟一眼——确认没动静后，终于撑不住了，脑袋一歪靠在闹钟上又睡着了。画面定格。音效：安静的滴答声中，蘑菇角色摇头时极轻的沙沙声，打哈欠的"嘟啊～"，最后靠上去时毛绒蹭金属的轻微声，呼吸声恢复"呼——嘟……"。只要音效，不要背景音乐，不要字幕。
