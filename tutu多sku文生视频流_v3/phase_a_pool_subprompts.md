# Phase A Pool Sub-Prompts

> 本文件由 `phase_a.py` 解析，切分成七个池子的子 prompt。解析规则：`## <pool>` 为池子名；如果池子下有 `### <key>`，则池子是 dict（如 `daily`、`weather`）；否则池子是整段字符串（如 `background`、`lifestyle`）。
>
> 这些子 prompt 不会单独发给 Gemini，而是作为每条 context 的 `reference_hints` 附在 blueprint 上，随 Phase A 的用户 prompt 一起传给 Gemini，用来指示"这个 slot / 这种天气应该怎么取向"。修改这份 md 可以直接调整 Phase A 里每个池子的写作指引。

## daily

### morning
你负责早晨时段的时间氛围。重点是清晨光线、空气、刚开始运转的感觉，这只是气氛参考，不要把早晨强行写成刚醒或只允许低强度动作。

### late_morning
你负责上午偏后时段的时间氛围。重点是节奏已经展开、环境更明确、生活状态更进入轨道，但不要把它写成固定模板。

### afternoon
你负责下午时段的时间氛围。不要限制必须是哪一种。

### golden_hour
你负责傍晚暖光时段的时间氛围。不要默认一定是收尾、回看或慢下来。

### night
你负责夜晚时段的时间氛围。不要默认必须休息、安静或准备睡觉。

## weather

### 晴
你负责晴天条件。重点思考光线、阴影、暖感、晒到一小块太阳、空间清透这些气质。

### 风
你负责有风条件。重点思考空气流动、边缘轻晃、被风影响站位、、轻微不稳、联想到别处，不要写成夸张灾难感。

### 雨
你负责下雨条件。重点思考下雨、水痕、声音，不要写成情绪过重的苦情戏。

### 阴
你负责阴天条件。重点思考潮气、慢节奏、低刺激的生活片段。

### 热
你负责偏热条件。重点思考找阴凉、减少大动作、懒一点、放慢一点、靠近凉一点的位置。

## weather_default

天气只作为倾向，不要变成唯一决定因素。

## background

你负责 background 池。请只定义背景环境和生活语境：包括空间状态、光线、温度、材质、季节、时间氛围。不要直接写动作，不要给出可照搬的短语清单。

**空间维度必须发散**——别把蘑菇TUTU 永远关在屋里桌面/窗台。下面只是**参考方向，绝不限定，鼓励大胆延展到列表之外**：

- **居家室内**：客厅、卧室、书房、厨房、餐厅、卫生间、玄关、阳台、储物间、楼梯间……
- **居家细节角落**：抽屉里、书架上、衣柜里、被窝里、冰箱内、洗手台、镜柜、画框边……
- **户外公共场所**：街道、公园、咖啡馆、商店橱窗、市场、车站、广场、博物馆门口、电影院、图书馆……
- **户外自然**：花园、菜园、山林、海边、溪流、湖畔、雪地、雨中、田野、花丛、池塘、草坪、沙地……
- **旅途/移动场景**：行李箱、汽车里、车窗边、火车上、机舱、酒店房间、民宿、地图上……

以上仅作启发参考，请基于 slot/weather/season 自由想象任何合适的真实生活空间，越出列表也欢迎。

写 background 时**必须配合 slot/weather/season** 给出当下情境的细节（光线方向、温度感、风的强弱、空气湿度、声音背景），不要给笼统标签。

## lifestyle

你负责 lifestyle 池。请只定义这条视频的生活质感——是哪种"过日子"的感觉。不要把 lifestyle 写成动作，也不要写成空泛标签。

可以是任何生活质感：晨起仪式、忙碌中的小喘息、午后慵懒、傍晚归来、夜晚独处、节日热闹、旅途新鲜、风雨天的躲避感、雪天的安静感、运动出汗的清爽、做饭的烟火气、工作专注、读书沉浸、听音乐放空、户外散步舒展、旅游到处看看、节令仪式（春节/中秋/生日/雨水节气）、季节交替的过渡感等。

## action

你负责 action 池。请只定义秃秃这条视频里在做的**一件具体、轻量、可拍、可视化的小事**。

### 创作方向：三种模式混合输出（重要，比例 ≈ A:B:C = 6:3:1）

秃秃在视频里的"身份/状态"分三种模式，单批必须按这个比例**混合**：

**模式 A — 真的在像人一样生活做事**（单批 **≈60%**，**首选模式**）
秃秃就是它本人，但**像人一样真的在做某项真实生活/户外活动**。它的身份可以是户外爱好者、运动爱好者、做家务的居家者、各种工作岗位上的人，但**它不是在假扮**——它就是在做这件事，所有场景和工具都是该活动的真实场景和真实工具的微缩版。
- 写法可以是"秃秃在 [真实活动场景] [做该活动]"或"秃秃是一名 [身份]，在 [真实场景] [做活动]"
- 真实活动场景泛指：户外的山/水/路/林/雪/雨/沙/海等真实自然环境；公园/街道/咖啡馆/市场/车站/办公室等真实公共场所；家中各功能区；运动场/游泳池/游乐场等专业活动场所；旅途中的车船机舱/景点；以及各种真实工作岗位现场。**鼓励大胆发散到任何真实生活/户外活动**，不要总锁在桌面/窗台。
- **绝对禁止**：把家居/日常小物件改造成与该活动不相关的代用道具（如树叶当冲浪板、瓶盖当方向盘、吸管当桨等）；这是 B 模式特征，不属于 A
- **绝对禁止**："模仿/扮演/假装/装作/像 X 一样/把 X 当作 Y/X 当 Y 用/X 充当 Y" 等词

**模式 B — 模仿人，用一些道具模仿**（单批 **≈30%**）
秃秃**煞有介事地假扮**某个角色——既可以扮演真实生活中的职业（厨师、外卖员、画家等），**也可以扮演虚构/想象/虚拟世界里的角色**（比如施展魔法的人、武侠世界的人、未来世界的人、童话角色、科幻角色等任意想象身份）。
- 典型特征：把家居/日常小物件改造成与该角色相关的代用道具，画面有戏剧化反差萌
- **必须包含**"模仿/扮演/假装/装作/像 X 一样"等明确表示假扮的词

**模式 C — 还是它本人，纯萌物自由日常**（单批 **≈10%**，少量即可）
秃秃就是它自己（4cm 小蘑菇），随意地做点小事——发呆、打哈欠、伸懒腰、追自己的影子、闻一闻、晃一晃，没有明确身份，也不写"模仿"。

**A 与 B 的核心区分（最关键的一条铁律）**：
- **A 是真做事 → 必须用该活动真实专业的道具/工具**（虽然是微缩尺寸的，但形态和功能是该专业真品。例如：园丁用真小铲子，画家用真笔，厨师用真小刀/勺子，攀岩家用真小绳/真小冰镐/真小岩点）
- **B 是模仿 → 必须用其他家居/日常物件来模仿**（把现成的家居小物件改造成代用道具。例如：模仿冲浪者用西瓜皮当板，模仿撑伞绅士用瓶盖当伞，模仿宇航员用果冻壳当头盔）
- **不能交叉**：A 用代用道具就是错的（应改为 B），B 用真实专业道具就缺了 cosplay 反差萌（应改为 A）
- 同一个职业身份在两种模式下都可以出现，区别只在于场景/工具是否真实匹配
- 单批输出**不许有任何条目混合 A 与 B 的特征**——一条只能属于一种模式

**A 模式禁忌写法（这些是离谱组合，绝对不要照搬）**：
- ❌ "作为真正的排水工，在咖啡杯垫边缘修筑挡水堤坝" → 排水工不会去杯垫干活，且"塑料片当堤坝"是 B 写法
- ❌ "作为真正的攀岩家，抠帐篷铝合金地钉防滑纹" → 攀岩家应该在岩壁，不是地钉
- ❌ "像真正的晨跑者，沿长椅边缘的缝隙跑步" → 晨跑应该在跑道/小路
- ❌ "作为真正的航海家，站在树叶上破浪前行" → 树叶当船是 B 写法
- 正确做法：A 模式写时，请把场景定位在该活动**真实会发生的地方**，工具用该专业**真品的微缩版**

**模式 B — cosplay 模仿/扮演**（建议 20–40%，**不要超过 40%**）
秃秃**煞有介事地假扮**某个角色——典型特征是把家居/日常小物件改造成与该角色相关的代用道具，画面有戏剧化反差萌。**写法上必须包含"模仿/扮演/假装/装作/像 X 一样"等明确表示假扮的词**。

**模式 C — 纯萌物自由日常**（≈20–30%）
秃秃就是它自己（4cm 小蘑菇），随意地做点小事——发呆、打哈欠、伸懒腰、追自己的影子、闻一闻、晃一晃，没有明确职业身份，也不写"模仿"。

**A / B / C 区分铁律（务必遵守）**：
- 一条 action_theme **只能属于一种模式**，不允许混合
- **A 不准混入 B 的句式**：禁止"此刻正是真正的 X"+"用 XX 当 YY"组合——这是把 B 包装成 A，是错误写法
- **B 不准伪装 A**：B 必须有"模仿/扮演/假装"等词，让读者一眼看出是 cosplay
- **C 不要硬塞职业身份**：C 就是它本来的样子，不需要给一个角色帽子

**以下只是参考方向，绝不限定，鼓励大胆发散到列表之外的任何角色 / 场景 / 动作**：

- **各类职业**（参考，请大胆扩展）：上班族、外卖员、快递员、厨师、医生、护士、老师、学生、警察、消防员、画家、程序员、设计师、摄影师、咖啡师、农夫、园丁、清洁工、导游、健身教练、修车工、空乘、列车员、检票员、图书管理员、街头艺人、宇航员、潜水员、考古学家、侦探、记者、主持人、调酒师、收银员、博物馆讲解员、宠物美容师、海关、面试官、辅导员、瑜伽老师、马戏团演员、魔术师、邮递员……（请发散到任何合适的真实职业或想象身份。**注意：秃秃自己的性格保持不变（好奇、胆小但鼓起勇气、犯懒犯困、认真但常搞砸），扮演只是它在"假装做一个职业"，不是把它变成另一个角色**）
- **一天的工作/学习节奏**（参考，可任意延展）：早起、刷牙、早餐、通勤、开会、午休、加班、下班、聚会、夜读、洗澡、入睡；上学、写作业、考试、课间、放学、社团活动……
- **衣食住行**（参考）：穿搭、洗漱、备餐、烹饪、布置家居、出行、收拾、搬家
- **生活仪式**（参考）：生日、节日、纪念日、婚礼路过、毕业、新年、看演出、看球赛
- **运动/健身/旅游/休闲娱乐**（参考）：跑步、瑜伽、游泳、爬山、看书、听音乐、看电影、追剧、玩游戏、骑行、滑板、钓鱼
- **天气/季节小事件**（参考）：避雨、迎风、晒衣、躲阳光、踩水洼、追雪花、捡落叶、采春芽

以上**全部只是启发参考，请你发散到任何合适的真实生活片段**——列表里没出现的角色/职业/动作/场景同样欢迎，越奇越好。

### 空间维度也要彻底打开

- **室内**：客厅、卧室、书房、厨房、餐厅、卫生间、玄关、阳台、楼梯、储物间、车库、衣帽间……

- **户外公共**：街道、公园、咖啡馆、商店橱窗、市场、车站、广场、博物馆、电影院、图书馆、办公楼走廊……
- **户外自然**：花园、菜园、山林、海边、溪流、湖畔、雪地、雨中、田野、花丛、池塘、沙地……
- **旅途/移动**：行李箱、汽车里、车窗边、火车上、机舱、酒店房间、民宿、地图上……

### 多样性硬约束（这是关键）

- 同一批 **action 主动词不重复**（任意两条不能围绕同一个核心动作）
- 同一批 **互动物体不重复**（不要两条都围绕"露珠""曲别针""碎屑""鞋带"等同类小物件）
- 同一批 **主场景大类不超过 2 条**（仅作参考的几大类："室内 / 室内细节角落 / 户外公共 / 户外自然 / 旅途"，但**不限于此**，鼓励发散到更多空间）
- 单批至少 **30% 户外/公共/自然/旅途**——不要全锁室内
- **三种身份模式必须按 6:3:1 比例混合**：
  - 模式 A（真的在像人一样生活做事，真实场景+真实工具）：单批 **≈60%**（首选模式）
  - 模式 B（模仿人/虚构角色，用日用品当道具假扮）：单批 **≈30%**
  - 模式 C（还是它本人，纯萌物自由日常）：单批 **≈10%**（少量即可，主要负担是 A）
- **禁止机械句式**：不要每条都"秃秃正努力地 X 着 Y，试图把它 Z 到..."、"费力地 X 着...，想把...挪到..."这种模板；每条用不同句式、不同动词、不同语序起头
- 不要只写"站在/停在某处观察/看/被吸引/好奇地靠近"——这不算有效动作

### 防重复约束（跨批次/跨整个项目）

- 一条 action_theme 一旦在历史里出现过（同一批或之前批次），后续坚决不再写——哪怕换一个表面的物体也不行（"推露珠到多肉"和"推水珠到薄荷叶"算同一条）
- 用户消息里如果给出了"已生成过的 action_theme 列表"作为反例，本次输出**严禁**与列表中任何一条在动词或物体上发生明显重复
- 写作时**先从冷门角度想**，再去填具体物体；不要从最常见的"推/挪/搬一个小东西"想起

**领域词只是大类提示，不要逐字照抄**，请你结合 slot/weather/season 自由组合一件具体小事。

**action_theme 字段格式禁忌**：
- 禁止把"模式 A："/"模式 B："/"模式 C："/"(A)"/"cosplay："/"模式："等元数据分类标签写进 action_theme——它们是给你看的提示，不是输出内容
- 禁止把上面这些指令性句子（如"借用灯光、街道或室内外反差..."、"包含它对某个日常物体的互动"）当成 action_theme 内容；这些是写作方向，**不能原文复制**
- action_theme 必须是一个**具体的小事件描述**，以"秃秃"（或"它"）为主语，紧接动作 + 物体 + 情境，**不要带 meta 标签前缀**

## mood

你负责 mood 池。请只定义蘑菇TUTU的情绪。不要写成大起大落的戏剧冲突，也不要写成情绪表演。

## guardrail

你负责 guardrail 池。请确保时间、天气、动作、角色设定、情绪彼此合理；避免危险、攻击、恐怖、成人化、重复模板化；输出必须适合批量生产且低相似度。
