# Fast 版口语说明

这套 `agent_fast` 就是“不要文生图，直接文生视频”的实验路线。

原来的路线是：

```text
A1 -> 文生图首帧 -> 图生视频
```

fast 版直接变成：

```text
A1 context -> 文生视频 prompt -> 文生视频
```

所以 Fast Phase C 不再看任何文生图中间层。它只看 A1 context 里的 background、lifestyle_theme、action_theme、mood_theme，然后直接写 Seedance 文生视频 prompt。

角色参考图只是在提交 Seedance 时一起传过去，用来辅助 TUTU 外观稳定。它不是首帧，也不是构图参考。

这条路线的优点是快，少了文生图 prompt 和首帧图两步。缺点是没有首帧图锁构图，所以场景、道具和开场画面会更依赖文生视频 prompt 本身。

建议先跑 3 条测试，看 TUTU 外观稳不稳、动作够不够明显、场景有没有乱。
